Portable setup

Starter Kit — run in a fresh session

Clone the repo and drop this into Claude Code. Everything runs natively — no Python, no Node.js, no servers.

Option 1 — Clone from GitHub

git clone https://github.com/paigerogers-pixel/orion-seo-pipeline.git cd orion-seo-pipeline

Option 2 — Download ZIP

⬇ Download orion-seo-pipeline.zip

Then in Claude Code

Open the folder in Claude Code File → Open Folder → select orion-seo-pipeline
Copy .env.example to .env Add your ANTHROPIC_API_KEY (minimum required to run)
Run the pipeline Just say: run the Orion pipeline — all 6 agents fire automatically
Review outputs Outputs in output/ · Jira tickets created · Confluence auto-updated
